# kalkulator-sederhana

This 'kalkulator sederhana' is a web apps use HTML, CSS, and Javascript, contain just 1 page with 5 features :
1. Addition ( + ),
2. Substraction ( - ),
3. Multiplication ( * ),
4. Division ( / ), and
5. Clear input and result.
